Basic Progress

Standard gradient progress bar.

View Code
HTML 
<div class="progress">
  <div class="progress-bar" style="width: 65%;"></div>
</div>
CSS 
.progress {
  width: 100%;
  height: 8px;
  background: rgba(255, 255, 255, 0.1);
  border-radius: 4px;
  overflow: hidden;
}

.progress-bar {
  height: 100%;
  background: linear-gradient(90deg, #667eea, #764ba2);
  border-radius: 4px;
  transition: width 0.3s ease;
}

Animated Stripes

Moving stripe animation.

View Code
HTML 
<div class="progress progress-animated">
  <div class="progress-bar" style="width: 65%;"></div>
</div>
CSS 
.progress-animated .progress-bar {
  background-image: linear-gradient(
    45deg,
    rgba(255, 255, 255, 0.15) 25%,
    transparent 25%,
    transparent 50%,
    rgba(255, 255, 255, 0.15) 50%,
    rgba(255, 255, 255, 0.15) 75%,
    transparent 75%
  );
  background-size: 1rem 1rem;
  animation: progressStripes 1s linear infinite;
}

@keyframes progressStripes {
  from { background-position: 1rem 0; }
  to { background-position: 0 0; }
}

Indeterminate

Loading state with no known progress.

View Code
HTML 
<div class="progress progress-indeterminate">
  <div class="progress-bar"></div>
</div>
CSS 
.progress-indeterminate .progress-bar {
  width: 30% !important;
  animation: indeterminate 1.5s ease-in-out infinite;
}

@keyframes indeterminate {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(400%); }
}

Circular Progress

Round progress indicator.

65%
View Code
HTML 
<div class="progress-circular">
  <svg width="80" height="80">
    <circle class="progress-bg" cx="40" cy="40" r="35"></circle>
    <circle class="progress-fill" cx="40" cy="40" r="35" 
      style="stroke-dashoffset: 77;"></circle>
  </svg>
  <span class="progress-circular-label">65%</span>
</div>
CSS 
.progress-circular {
  width: 80px;
  height: 80px;
  position: relative;
}

.progress-circular svg { transform: rotate(-90deg); }

.progress-circular circle {
  fill: none;
  stroke-width: 6;
  stroke-linecap: round;
}

.progress-circular .progress-bg {
  stroke: rgba(255, 255, 255, 0.1);
}

.progress-circular .progress-fill {
  stroke: #667eea;
  stroke-dasharray: 220;
  stroke-dashoffset: 220;
  transition: stroke-dashoffset 0.3s ease;
}

.progress-circular-label {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  font-weight: 600;
  color: #fff;
}

Basic Tabs

Standard tab navigation with underline.

View Code
HTML 
<div class="kvo-tabs">
  <button class="kvo-tab active">Overview</button>
  <button class="kvo-tab">Features</button>
  <button class="kvo-tab">Pricing</button>
  <button class="kvo-tab">FAQ</button>
</div>
CSS 
.kvo-tabs {
  display: flex;
  gap: 4px;
  border-bottom: 1px solid rgba(255, 255, 255, 0.1);
}

.kvo-tab {
  padding: 12px 20px;
  color: rgba(255, 255, 255, 0.6);
  background: transparent;
  border: none;
  cursor: pointer;
  position: relative;
}

.kvo-tab.active {
  color: #fff;
}

.kvo-tab.active::after {
  content: '';
  position: absolute;
  bottom: -1px;
  left: 0;
  right: 0;
  height: 2px;
  background: linear-gradient(90deg, #667eea, #764ba2);
}

Toggle Switches

6 beautiful toggle switch styles for settings.

Switch Styles

  • iOS Style
  • Material
  • Neon
  • Gradient
  • Minimal
  • Square

iOS Style Switch

Classic iOS-inspired toggle switch.

View Code
HTML 
<label class="switch-ios">
  <input type="checkbox" />
  <span class="slider"></span>
</label>
CSS 
.switch-ios input:checked + .slider {
  background-color: #34c759;
}

.switch-ios input:checked + .slider::before {
  transform: translateX(22px);
}

Modern Radio

Clean radio button with smooth animation.

View Code
HTML 
<label class="radio-modern">
  <input type="radio" name="group" />
  <span class="radiomark"></span>
  <span class="label-text">Option 1</span>
</label>
CSS 
.radio-modern input:checked ~ .radiomark {
  background: #667eea;
  border-color: #667eea;
}

Gradient Radio

View Code
HTML 
<label class="radio-gradient">
  <input type="radio" name="plan" />

</label>

.radio-gradient .radiomark::before {
  background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
  opacity: 0;
}

.radio-gradient input:checked ~ .radiomark::before {
  opacity: 1;
}

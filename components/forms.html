Cyber Corner

Input dengan corner accents yang expand saat focus.

View Code
HTML 
<div class="input-cyber-group">
  <input type="text" class="input-cyber" placeholder="Username" />
</div>
CSS 
.input-cyber-group {
  position: relative;
  width: 100%;
}

.input-cyber-group::before,
.input-cyber-group::after {
  content: '';
  position: absolute;
  width: 8px;
  height: 8px;
  border: 2px solid #00d4ff;
  transition: all 0.3s;
  pointer-events: none;
}

.input-cyber-group::before {
  top: -2px;
  left: -2px;
  border-right: none;
  border-bottom: none;
}

.input-cyber-group::after {
  bottom: -2px;
  right: -2px;
  border-left: none;
  border-top: none;
}

.input-cyber-group:focus-within::before,
.input-cyber-group:focus-within::after {
  width: 16px;
  height: 16px;
}

.input-cyber {
  width: 100%;
  padding: 16px 20px;
  font-size: 0.9rem;
  color: #fff;
  background: transparent;
  border: 1px solid rgba(255,255,255,0.2);
  transition: all 0.3s;
}

.input-cyber:focus {
  outline: none;
  border-color: #00d4ff;
  box-shadow: 0 0 20px rgba(0, 212, 255, 0.2);
}

Glassmorphism

Input dengan efek glass blur modern.

View Code
HTML 
<div class="input-glass-group">
  <i class="fas fa-user input-glass-icon"></i>
  <input type="text" class="input-glass has-icon" placeholder="Username" />
</div>
CSS 
.input-glass {
  width: 100%;
  padding: 16px 20px;
  font-size: 0.9rem;
  color: #fff;
  background: rgba(255,255,255,0.05);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255,255,255,0.1);
  border-radius: 12px;
  transition: all 0.3s;
}

.input-glass:focus {
  outline: none;
  background: rgba(255,255,255,0.1);
  border-color: rgba(255,255,255,0.3);
  box-shadow: 0 8px 32px rgba(0,0,0,0.2);
}

.input-glass-icon {
  position: absolute;
  left: 16px;
  top: 50%;
  transform: translateY(-50%);
  color: rgba(255,255,255,0.4);
  transition: color 0.3s;
}

.input-glass-group:focus-within .input-glass-icon {
  color: #00d4ff;
}

.input-glass.has-icon {
  padding-left: 48px;
}

  font-size: 0.9rem;

Toggle Switch

Toggle switch dengan glow effect.

View Code
HTML 
<label class="toggle-switch">
  <input type="checkbox" checked />
  <span class="toggle-slider"></span>
</label>

<label class="toggle-switch toggle-switch-purple">
  <input type="checkbox" />
  <span class="toggle-slider"></span>
</label>
CSS 
.toggle-switch {
  position: relative;
  display: inline-block;
  width: 56px;
  height: 28px;
  cursor: pointer;
}

.toggle-switch input {
  opacity: 0;
  width: 0;
  height: 0;
}

.toggle-slider {
  position: absolute;
  inset: 0;
  background: rgba(255, 255, 255, 0.1);
  border-radius: 28px;
  transition: all 0.3s;
}

.toggle-slider::before {
  content: '';
  position: absolute;
  width: 22px;
  height: 22px;
  left: 3px;
  bottom: 3px;
  background: #fff;
  border-radius: 50%;
  transition: all 0.3s;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.3);
}

.toggle-switch input:checked + .toggle-slider {
  background: #00d4ff;
  box-shadow: 0 0 20px rgba(0, 212, 255, 0.4);
}

.toggle-switch input:checked + .toggle-slider::before {
  transform: translateX(28px);
}

Checkbox

Custom checkbox dengan glow effect.

View Code
HTML 
<label class="checkbox-cyber">
  <input type="checkbox" checked />
  <span class="checkmark"><i class="fas fa-check"></i></span>
  Remember me
</label>
CSS 
.checkbox-cyber {
  display: flex;
  align-items: center;
  gap: 12px;
  cursor: pointer;
  color: rgba(255,255,255,0.8);
  font-size: 0.9rem;
}

.checkbox-cyber input {
  display: none;
}

.checkbox-cyber .checkmark {
  width: 22px;
  height: 22px;
  border: 2px solid rgba(255,255,255,0.3);
  border-radius: 4px;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: all 0.3s;
}

.checkbox-cyber .checkmark i {
  color: #000;
  font-size: 0.7rem;
  opacity: 0;
  transform: scale(0);
  transition: all 0.2s;
}

.checkbox-cyber input:checked + .checkmark {
  background: #00d4ff;
  border-color: #00d4ff;
  box-shadow: 0 0 15px rgba(0, 212, 255, 0.4);
}

.checkbox-cyber input:checked + .checkmark i {
  opacity: 1;
  transform: scale(1);
}

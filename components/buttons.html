Cyber Corner

Button dengan corner accents yang expand saat hover.

View Code
HTML 
<button class="btn-cyber">LAUNCH</button>
<button class="btn-cyber btn-cyber-pink">EXPLORE</button>
CSS 
.btn-cyber {
  position: relative;
  padding: 16px 48px;
  font-size: 0.9rem;
  font-weight: 600;
  color: #fff;
  background: transparent;
  border: 1px solid rgba(255,255,255,0.3);
  cursor: pointer;
  text-transform: uppercase;
  letter-spacing: 3px;
  overflow: hidden;
  transition: all 0.3s;
}

.btn-cyber::before,
.btn-cyber::after {
  content: '';
  position: absolute;
  width: 8px;
  height: 8px;
  border: 2px solid #00d4ff;
  transition: all 0.3s;
}

.btn-cyber::before {
  top: -2px;
  left: -2px;
  border-right: none;
  border-bottom: none;
}

.btn-cyber::after {
  bottom: -2px;
  right: -2px;
  border-left: none;
  border-top: none;
}

.btn-cyber:hover {
  background: rgba(0, 212, 255, 0.1);
  border-color: #00d4ff;
  box-shadow: 0 0 20px rgba(0, 212, 255, 0.3);
}

.btn-cyber:hover::before,
.btn-cyber:hover::after {
  width: 20px;
  height: 20px;
}

  content: '';

  width: 6px;
  height: 6px;
  background: #00d4ff;
  border-radius: 50%;
  top: -3px;
  left: 20%;
  animation: orbit-top 3s linear infinite;
  box-shadow: 0 0 10px #00d4ff;
}

.btn-orbit::after {
  content: '';
  position: absolute;
  width: 6px;
  height: 6px;
  background: #ff0080;
  border-radius: 50%;
  bottom: -3px;
  right: 20%;
  animation: orbit-bottom 3s linear infinite;
  box-shadow: 0 0 10px #ff0080;
}

@keyframes orbit-top {
  0% { left: 10%; }
  50% { left: 80%; }
  100% { left: 10%; }
}

@keyframes orbit-bottom {
  0% { right: 10%; }
  50% { right: 80%; }
  100% { right: 10%; }

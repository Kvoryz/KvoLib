Modern Checkbox

Smooth checkbox with scale animation.

View Code
HTML 
<label class="checkbox-modern">
  <input type="checkbox" />
  <span class="checkmark"></span>
  <span class="label-text">Accept terms</span>
</label>
CSS 
.checkbox-modern input:checked ~ .checkmark {
  background: #667eea;
  border-color: #667eea;
  transform: scale(1.05);
}

3D Effect Checkbox

Neumorphic 3D style checkbox.

View Code
HTML

Nav Style Link

Navigation link with scaling underline.

Home Products About Contact

HTML

CSS

  color: rgba(255, 255, 255, 0.6);
  text-decoration: none;
  padding: 8px 0;

  position: absolute;

  transform: scaleX(1);
}

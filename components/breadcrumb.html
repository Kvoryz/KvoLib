Basic Breadcrumb

Standard breadcrumb navigation.

Home Products Category
View Code
HTML 
<nav class="kvo-breadcrumb">
  <a href="#" class="kvo-breadcrumb-item">Home</a>
  <span class="kvo-breadcrumb-separator"><i class="fas fa-chevron-right"></i></span>
  <a href="#" class="kvo-breadcrumb-item">Products</a>
  <span class="kvo-breadcrumb-separator"><i class="fas fa-chevron-right"></i></span>
  <span class="kvo-breadcrumb-item active">Category</span>
</nav>
CSS 
.kvo-breadcrumb {
  display: flex;
  align-items: center;
  gap: 8px;
  font-size: 0.875rem;
}

.kvo-breadcrumb-item {
  color: rgba(255, 255, 255, 0.5);
  transition: color 0.2s;
}

.kvo-breadcrumb-item:hover { color: #fff; }
.kvo-breadcrumb-item.active { color: #fff; }
